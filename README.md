# Kakao Chat Integration Plugin

Integrate Kakao chat system with Discourse.

This is a fork of the official [discourse-chat-integration](https://github.com/discourse/discourse-chat-integration) plugin, maintained by [AirflowKorea](https://github.com/AirflowKorea/kakao-discourse-chat-integration).

For more information, please see: https://meta.discourse.org/t/chatroom-integration-plugin-discourse-chat-integration/66522 (Upstream documentation)
